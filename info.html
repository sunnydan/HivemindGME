The Hivemind GM Emulator v1.8 HGME v1.5
Info
  • Type in a room name and a username for yourself, and click "Join Room" to begin. If a room with that name does not already exist, one will be created.
  • The section of the interface with a green border is the "active" part of the interface. You cannot interact with the inactive part of the interface.
  • Type in a question and click or press Enter to pose a question to the Hivemind GM. All the players in the room will then get a chance to submit an answer. An answer will be randomly selected among all those given to be the final answer, which the system will respond with.
  • The system recognizes 3 types of question: Yes or No questions, Numeric questions, and Open-ended questions. Each of these types of question has their own answering interface to make providing an answer as easy as possible.
    • Yes or No questions begin with: "am", "is", "do", "are", "can", "did", "has", "may", "was", "does", "have", "were", "will", "could", "would", or "should."
    • Numeric questions begin with: "how many", "how much", or "how few"
    • All other entered questions are assumed to be Open-ended.
  • If you want to ask a question like, "How far is it to the next town?" and want to use the Numeric answering interface, instead phrase your question like, "How many miles is it to the next town?"
  • The first player to submit an answer to any question will start a timer. A final answer will be selected upon the timer running out, regardless of how many players have submitted an answer.
    • Yes or No questions have a default timer of 10 seconds.
    • Numeric questions have a default timer of 20 seconds.
    • Open-ended questions have a default timer of 30 seconds.
  •  The system will respond with "No Answer" if no answers have been submitted by the time the timer runs out, or if all players in the room Abstain from submitting an answer.
  • Enabling the "Allow players to make statements?" setting will allow players to post 'statements' directly to the log.
  • There is a small chance that Something Unexpected will happen after any answer. The chance of this happening can be turned off or configured in the settings.
  • For groups of players that are on a voice call or in the same room as each other, there is "Live Play Mode". Live play mode assumes that every player hears the question spoken aloud, and allows you to simply press a button with the type of question you think it is to start the answer poll. Typed questions will also not show who asked them in this mode, so that players can ask questions without having to speak it aloud if they wish.
